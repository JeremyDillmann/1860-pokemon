# 1860 Pokemon

Game Boy Color-style browser RPG celebrating TSV 1860 München history.

**Play now:** [jeremy-sax.com/1860pokemon](https://jeremy-sax.com/1860pokemon)

## Features
- 4 arenas spanning 1848 to today
- Turn-based battles with collectible historical players
- Type system and moral choices
- Authentic GBC aesthetic: 320x288 canvas, 16x16 tiles, Web Audio 8-bit BGM
- Single HTML file, no dependencies

## Tech
- Canvas + Web Audio API
- Vanilla JS, zero dependencies
- ~47KB single file
